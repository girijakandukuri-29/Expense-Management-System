# 💰 Smart Personal Finance Manager

An AI-powered Expense Management System that automates personal finance tracking using OCR receipt scanning, SMS-based transaction parsing, LSTM-based budget prediction, and an intelligent AI financial assistant.

---

# 📖 Project Overview

Smart Personal Finance Manager is a full-stack web application designed to simplify personal expense management by automatically collecting expenses from multiple sources and providing intelligent financial insights.

Unlike traditional expense trackers that require manual data entry, this system combines physical receipt scanning and digital transaction tracking to provide a complete financial overview.

---

# ✨ Features

- 🔐 Secure User Registration & Login
- 💸 Add, Edit & Delete Expenses
- 📷 OCR-based Receipt Scanner using Tesseract OCR
- 📱 Automatic SMS Parsing for UPI & Bank Transactions
- 📊 Interactive Dashboard with Expense Analytics
- 🥧 Monthly Reports & Category-wise Pie Charts
- 🤖 AI Financial Chatbot for Personalized Suggestions
- 📈 LSTM-based Monthly Budget Prediction
- 💾 MongoDB Database Integration
- 🔍 Smart Expense Categorization
- 📑 Expense History
- 🔒 Secure Authentication & Data Storage
